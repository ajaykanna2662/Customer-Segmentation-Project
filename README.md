# Customer Segmentation Project

## Project Description
This project performs customer segmentation using K-Means clustering based on Annual Income and Spending Score.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Customer segmentation
- Data visualization
- Customer behavior analysis
- Clustering using K-Means

## Output
Customers are divided into different groups and visualized using graphs.

## Expected Outcome
Gain experience in customer analytics, segmentation, and targeted insights.
